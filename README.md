# PracticaPruebasUnitarias

Nombre: Melany Lema

Esta práctica implementa la clase SmartWallet en Java 17 con Maven y pruebas unitarias en JUnit 5, validando depósitos y retiros según las reglas de a práctica: monto positivo, límite de saldo para usuario Standard, cashback del 1% para depósitos mayores a 100, validación de saldo insuficiente y cambio a cuenta inactiva cuando el saldo llega a 0.

## Ejecutar pruebas

Comando:

mvn test
